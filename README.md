# to-do-list-app
A lightweight desktop to-do list app built with HTML, CSS, and JavaScript, packaged as a native Windows app using Electron. Features a clean dark UI, persistent task saving, a custom image banner, and a frameless window design.

Features

- Add, check, and delete tasks
- Tasks persist after closing the app (saved to todos.json)
- Custom image banner at the top
- Frameless window with drag-to-move and a close button
- Fully customizable colors and fonts via CSS variables
